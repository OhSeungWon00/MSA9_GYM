## 📚 **주제 선정 및 배경**
- 헬스장 이용자들이 효율적으로 **출입 및 예약 관리**를 할 수 있는 시스템이 필요함  
- **QR코드 기반 출입 시스템**, 예약, 트레이너 관리, 스케줄 관리 등 **핵심 기능을 통합한 웹 애플리케이션 제작**
